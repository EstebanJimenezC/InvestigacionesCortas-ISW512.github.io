Usabilidad Web

La usabilidad web es la facilidad que se le brinda a un usuario para navegar de una forma rápida, fácil e intuitiva, con diferentes aspectos como lo son el diseño gráfico, presentación y organización de la información, y en el funcionamiento mismo, una buena usabilidad web va a aumentar el porcentaje de usuarios satisfechos y a disminuir el abandono hacia nuestra página web.





Importancia

La usabilidad web es uno de los aspectos en los que más debemos de pensar a la hora de desarrollar nuestra página web, debido a que si ofrecemos una página fácil de utilizar, nuestros clientes estarán más sastifechos y les tomará menos tiempo realizar sus necesidades, una mala usabilidad es una de las principales características del abandono web,por lo tanto si desarrollamos páginas web lentas, díficiles de navegar y poco intuitivas no estaremos perdiendo solamente un usuario sino que perderíamos un pontencial cliente y una compra menos en nuestra página web.





Principios de la usabilidad web


Nombre Funcionalidad Ejemplo
Eficiencia Trata de que los usuarios puedan cumplir con sus necesidades en el menor tiempo y con la mayor facilidad posible. MS Exchange ofrece al usuario una configuración avanzada.
Eficacia La página web que desarrollemos debe ser rápida e intuitiva, acelerando los procesos que nuestros usuarios realicen. La página web del BAC es una página rápida, confiable y fácil de usar, en comparación a páginas del mismo segmento.
Satisfacción Busca que los usuarios tengan una experiencia óptima dentro de la página, de esto depende si el cliente vuelve a visitarnos. Amazon tiene una página web muy completa y confiable, mucho de su éxito es su sencillez, lo que atrae y satisface a sus usuarios.
Visibilidad del estado del sistema El sistema siempre debe mantener informado al usuario de lo que está ocurriendo. Al subir un archivo a Google Drive, el sistema nos indica que se está cargando y el tiempo restante.
Prevención de errores Evita que el usuario realice una acción y se genere un error. El autofill que se nos muestra al buscar en algunas páginas web.
Diseño estético y minimalista La información y el diseño de nuestra página web debe contener solo necesario. EYRC es una página web con mucho minimalismo, se basa en una tipografía clara y el ofrecimiento de sus productos de una manera sencilla y concisa.
Ayuda y documentación Debemos desarrollar una página web fácil de usar, pero siempre es bueno ofrecerle al usuario una guía rápida y efectiva. Las pequeñas guías de uso que nos ofrecen algunas páginas web como Canva.
Control y libertad del usuario Los usuarios cometen muchos errores a la hora de navegar en las páginas web, debemos dejar una salida clara, para que puedan volver con fácilidad. Funciones de deshacer y rehacer, botones de retroceso.
